<TopBlock>

Prisma Data Platform supports the following roles for team members on projects:

- **Project owner**
- **Admin**
- **Developer**
- **Collaborator**
- **Viewer**

To add users, you need to use their GitHub username and they need to have an account in the platform.

</TopBlock>

## Roles and permissions

The table below lists the permissions for every supported role in the platform.

| Feature                            | Project owner | Admin | Developer | Collaborator | Viewer |
