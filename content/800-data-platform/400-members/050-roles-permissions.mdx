<TopBlock>

Prisma Data Platform supports role-based access (RBAC) to projects.

With the exception of **Project owner**, you can use the roles listed below to assign to team members and control which parts of the project they can only view or edit.

- **Project owner**
- **Admin**
- **Developer**
- **Collaborator**
- **Viewer**

To add users, you need to use their GitHub username and they need to have an account in the platform.

</TopBlock>
